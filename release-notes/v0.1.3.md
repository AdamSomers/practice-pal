# v0.1.3 — April 6, 2026

## Bug Fixes
- **Progress API crash fixed**: The SQL query used `ANY()` with a JavaScript array, which Drizzle ORM doesn't support. Replaced with `inArray()`. Previously the progress endpoint returned 500 on every request.
- **Streak calculation fixed**: Day comparison used PostgreSQL `date()` which returns a Date object, not a string. Now uses `to_char()` for reliable string comparison. Also fixed streak to count from yesterday if no practice today.
