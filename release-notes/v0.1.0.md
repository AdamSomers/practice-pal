# v0.1.0 — April 5, 2026

First tagged release of PracticePal.

## New Features
- **Repertoire sections practice** — Practice a piece broken into sections, each with its own reps and optional labels (e.g., measure ranges)
- **Printable charts** — Print icon on each chart opens a clean, printer-friendly view with items and empty checkboxes
- **Chart management** — Delete and duplicate charts from the Charts tab. Duplicate practice items in the chart builder.
- **Custom scale types** — "Other" option in scale types lets you enter any scale name
- **Cadence key type** — Major/minor key type selector added to cadences
- **About page** — Info button in the header shows app info and release notes
- **Inline custom rewards** — Create custom rewards directly in the goal creation flow
- **Reward settings on Goals & Progress** — Emoji category preferences and custom rewards managed from the Goals & Progress page instead of Settings

## Improvements
- Practice mode now shows all item details (key, type, composer, notes, modifiers, etc.)
- Composer auto-populates when entering a previously used piece title
- Responsive chart card layout (stacked on mobile, horizontal on desktop)
- Delete confirmations on charts, practice items, goals, and custom rewards
- Renamed "Progress" to "Goals & Progress"

## Bug Fixes
- Fixed empty state ("No Studios Yet") appearing when server session expires. App now redirects to login on 401.

## Cleanup
- Removed unused mobile tabs (Practice, Progress)
- Removed Mastery and Members tabs (covered by Settings)
- Removed measures field from repertoire (covered by sections)
