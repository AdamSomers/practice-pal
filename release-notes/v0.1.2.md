# v0.1.2 — April 6, 2026

## Bug Fixes
- **Progress page now shows data**: API was returning the wrong data shape (totalSeconds instead of totalPracticeMinutes, weekly buckets instead of daily). Now returns minutes, current streak, and Mon-Sun daily breakdown matching the frontend.
- **Chart item buttons always visible**: Edit, duplicate, and delete buttons on practice items were hidden behind hover on tablets and desktops. Now always visible.
- **Instant checkbox response**: Checking/unchecking practice items was blocked on the network round-trip, causing multi-second lag on slow connections. Now uses optimistic updates with rollback on failure.
